Vapour: a web-based validator tool to check the compliance of a web site against the recipes described in the Best Practices for Publishing RDF Vocabularies

Vapour is a web-based validator tool to check the compliance of a web site against the recipes described in the Best Practices for Publishing RDF Vocabularies (W3C Working Draft).

Please report bugs to diego.berrueta@fundacionctic.org. This work is not endorsed by W3C.
