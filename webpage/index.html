vapour

a web-based validator tool to check the compliance of a web site against the recipes described in the Best Practices Recipes for Publishing RDF Vocabularies

Vapour is a web-based validator tool to check the compliance of a web site against the recipes described in the Best Practices Recipes for Publishing RDF Vocabularies (W3C Working Draft).

Try our demo at: http://idi.fundacionctic.org/vapour.

Please, send any comment to the development mailing list or report bugs to the project's bug tracker. This work is not endorsed by W3C.
